# 2026 Nordic-RSE Conference

Website for 2026 Nordic-RSE conference. Based on [Academic Workshop](https://aterenin.github.io/academic-workshop), a Zola theme designed for hosting a website for scientific workshop or scientific seminar series.

## Build a local preview

Easiest to use Apptainer/Singularity

``` bash
apptainer pull docker://ghcr.io/getzola/zola:v2.19.1
apptainer run zola_v0.19.1.sif serve
```
